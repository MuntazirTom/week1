Attributes

Attributes means add extra information in element. There are 3 types of Attributes.

  1. Core Attributes
    1. id
    2. title
    3. class
    4. style
  2. Internationalizion Attributes
    1. Dir> direction
    2. lang> language
  3. Gerneric Attributes
    1. align
    2. valign
    3. bg
    4. width
    5. hieght
